# OniGo
Unityで簡単なゲーム作ってみました．Chromeで動作確認しています．  
下記URLからどうぞ．  
https://kenchalros.github.io/OniGo/

## 操作方法
**↑**で前進．**←**，**→** で方向転換．**shift**(左)でダッシュします．

## ルール
- 鬼が追いかけてきます．制限時間内に捕まらないよう逃げてください．
  - 髪の赤い鬼はプレイヤーが問答無用で追いかけてきます．
  - 髪の黒い鬼は近づくと追いかけてきます．
- フィールドにはアイテムが落ちてます．
  - 赤：一定時間スタミナMax
  - 青：制限時間が10秒延長
  - 黄：point加算
